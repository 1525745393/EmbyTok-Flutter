# 非纯净模式：中央播放/暂停图标自动隐藏

## 摘要

在非纯净模式（`isAutoPlay=false`）下，将中央播放/暂停按钮从"暂停时持续显示"改为"切换图标 + 2 秒自动隐藏"。信息条（含全屏观看按钮、标题、进度条）保持已实现的"始终可见"行为不变。

## 当前状态分析

### 中央播放按钮现状

[video_page_item.dart#L1056-L1074](file:///workspace/frontend/lib/widgets/video_page_item.dart#L1056-L1074) `_CenterPlayButtonWrapper`：

```dart
@override

  final isPlaying = ref.watch(isPlayingProvider);
  // 仅在控制器已初始化且非播放状态时显示中央播放按钮
  if (controller != null && controller!.value.isInitialized && !isPlaying) {
    return CenterPlayButton(onPlay: onPlay);
  }
  return const SizedBox.shrink();

```

**问题**：
- 仅在 `!isPlaying` 时显示，且持续显示不隐藏
- 不区分纯净/非纯净模式
- `CenterPlayButton` 组件硬编码 `Icons.play_arrow`，无暂停图标

### 单击行为现状

[video_page_item.dart#L665-L676](file:///workspace/frontend/lib/widgets/video_page_item.dart#L665-L676) 非纯净模式单击直接 `_togglePlay()`，不触发任何 Timer。

### 已有可参考的自动隐藏模式

[video_page_item.dart#L91-L92](file:///workspace/frontend/lib/widgets/video_page_item.dart#L91-L92) `_controlsHideTimer` + `_controlsAutoHideSeconds = 3`：
- `_showControls()` 中先 cancel 再 setState，然后启动 Timer
- `_hideControls()` 中 cancel + setState false
- dispose、didUpdateWidget 非当前页时 cancel

## 提议变更

### 决策

1. **图标切换**：播放时显示暂停图标，暂停时显示播放图标
2. **自动隐藏时长**：2 秒
3. **作用范围**：仅非纯净模式（`!isAutoPlay`）应用
4. **显示触发时机**：单击屏幕切换播放/暂停状态后立即显示，2 秒后自动隐藏

/// 显示中央播放/暂停按钮并启动自动隐藏计时器（仅非纯净模式）
void _showCenterButton() {
  _centerButtonHideTimer?.cancel();
  if (!mounted) return;
  setState(() => _centerButtonVisible = true);
  _centerButtonHideTimer = Timer(
    const Duration(seconds: _centerButtonAutoHideSeconds),
    _hideCenterButton,
  );
}

/// 隐藏中央播放/暂停按钮
void _hideCenterButton() {
  _centerButtonHideTimer?.cancel();
  if (mounted) setState(() => _centerButtonVisible = false);
}
```

class _CenterPlayButtonWrapper extends ConsumerWidget {
  final VideoPlayerController? controller;
  final VoidCallback onPlay;
  final bool visible; // 由父组件控制显示状态（非纯净模式下的自动隐藏）
  final bool isAutoPlay;

  const _CenterPlayButtonWrapper({
    required this.controller,
    required this.onPlay,
    required this.visible,
    required this.isAutoPlay,
  });

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    // 纯净模式不显示中央按钮（由 VideoControls 控制条操作）
    if (isAutoPlay) return const SizedBox.shrink();
    // 非纯净模式：由 visible 状态控制显示
    if (!visible) return const SizedBox.shrink();
    if (controller == null || !controller!.value.isInitialized) {
      return const SizedBox.shrink();
    }
    final isPlaying = ref.watch(isPlayingProvider);
    return CenterPlayButton(onPlay: onPlay, isPlaying: isPlaying);
  }
}
```

## 假设与决策

### 假设

1. **信息条始终可见**：当前 `_resetInfoHideTimer` 实现已禁用自动隐藏，本次变更不影响此行为
2. **纯净模式行为不变**：纯净模式下通过 VideoControls 控制条操作，不显示中央按钮
3. **初始状态**：进入页面时 `_centerButtonVisible = false`，即不主动显示中央按钮，仅在用户单击切换播放/暂停时显示
4. **页面切换清理**：`didUpdateWidget` 非当前页时取消计时器并隐藏中央按钮（与 `_controlsHideTimer` 一致）

### 设计决策

1. **不使用 `_isPlayingSubscription` 触发显示**：仅在用户主动 `_togglePlay()` 时触发显示，避免播放状态被动变化（如缓冲结束自动恢复播放）时意外显示按钮
2. **复用 `_controlsHideTimer` 模式**：保持代码风格一致性，便于维护
3. **图标切换**：播放时显示暂停图标（用户可点击暂停），暂停时显示播放图标（用户可点击播放），符合直觉

## 验证步骤

### 手动验证

1. **非纯净模式**：
   - 进入 Feed 页面，视频自动播放，中央按钮不显示
   - 单击屏幕 → 显示暂停图标，2 秒后自动隐藏
   - 再次单击 → 显示播放图标，2 秒后自动隐藏
   - 信息条（含全屏观看按钮、标题、进度条）始终可见
2. **纯净模式**：
   - 单击屏幕 → 切换 VideoControls 控制条显示/隐藏
   - 中央按钮不显示
3. **页面切换**：
   - 滑动到下一页 → 中央按钮立即隐藏，Timer 取消
4. **退出页面**：
   - 退出 Feed → Timer 取消，无内存泄漏
