# EmbyTok Flutter 版 - 验证检查清单

---

## 验证类别 A: 播放控制手势

### A1. 单击播放/暂停
- [ ] 单击视频区域，视频从暂停 → 播放
- [ ] 单击视频区域，视频从播放 → 暂停
- [ ] 播放状态变化后底部信息条显示正确（暂停时显示，播放后 3 秒自动隐藏）

### A2. 长按 2 倍速
- [ ] 按住 500ms 后，视频以 2.0x 速度播放
- [ ] 屏幕右上角显示 "2x Speed" 徽章或类似提示
- [ ] 松开后恢复到 1.0x（未滑动调整时）
- [ ] 长按期间视频仍在播放

### A3. 长按 + 上下滑动调整倍速
- [ ] 长按期间向上滑动（deltaY < 0），倍速增加
- [ ] 长按期间向下滑动（deltaY > 0），倍速减少
- [ ] 倍速范围：0.5x - 5.0x
- [ ] 松手后倍速保持（永久倍速，不恢复）
- [ ] 当前倍速显示在倍速按钮上

### A4. 水平滑动快进/快退
- [ ] 水平向右滑动 > 20px，显示 "+N 秒" 徽章
- [ ] 水平向左滑动 > 20px，显示 "-N 秒" 徽章
- [ ] 松手后视频跳转到对应位置
- [ ] 跳转后进度条同步更新
- [ ] seekOffset = round(deltaX / 5) 秒（5像素 = 1秒）

### A5. 双击点赞
- [ ] 300ms 内双击视频区域，触发点赞动画
- [ ] 心形动画从双击位置弹出
- [ ] 点赞状态同步到 `widget.item.isFavorite` 状态
- [ ] 再次双击切换收藏状态

## 验证类别 B: 可拖拽进度条

### B1. 进度条显示
- [ ] 视频时长 > 3 分钟时，底部信息条下方显示进度条
- [ ] 视频时长 < 3 分钟时，不显示进度条
- [ ] 进度条颜色：已播放部分粉色（primaryPink）
- [ ] 进度条颜色：未播放部分半透明黑色（black26）
- [ ] 进度条高度：6px（显示区域）
- [ ] 进度条圆角：3px

### B2. 点击跳转
- [ ] 点击进度条左侧 1/4 处，视频跳转到 25% 位置
- [ ] 点击进度条中点，视频跳转到 50% 位置
- [ ] 点击进度条右侧 3/4 处，视频跳转到 75% 位置
- [ ] 点击后进度条同步更新

### B3. 水平拖拽调整
- [ ] 从进度条 25% 处开始按住，拖拽到 75%，视频跳到 75%
- [ ] 拖拽过程中视频实时更新位置
- [ ] 拖拽结束后视频继续播放或保持暂停

### B4. 时间显示
- [ ] 进度条右侧显示当前时间（如 "12:34"）
- [ ] 显示总时长（如 "/ 45:00"）
- [ ] 时间格式：小于 1 小时显示 mm:ss，大于等于 1 小时显示 h:mm:ss
- [ ] 时间随播放位置实时更新

### B5. 5 秒自动隐藏
- [ ] 播放开始时显示进度条
- [ ] 5 秒内无操作，进度条淡出隐藏
- [ ] 用户点击/滑动后进度条重新显示，计时重置
- [ ] 暂停时进度条保持显示
- [ ] 隐藏动画：300ms fade out
