# EmbyX UI 结构分析与借鉴 Spec

## Why

EmbyTok 项目需要优化 UI 设计，参考 EmbyX 项目的成功设计模式可以提升用户体验。EmbyX 是一个 TikTok 风格的 Emby/Jellyfin Web 播放器，具有优秀的沉浸式界面设计。

## What Changes

- 分析 EmbyX 的 UI 布局结构和设计模式
- 对比 EmbyTok 当前的实现差异
- 提出可借鉴的设计改进建议

## Impact

- Affected specs: 视频播放界面、手势交互、响应式设计
- Affected code: `frontend/lib/widgets/` 目录下的 UI 组件

### 2. 核心设计特点

#### 2.1 颜色主题
- **主色**：`#52B54B`（Emby 绿）- 用于关注按钮、进度条
- **强调色**：`#EE3152`（抖音红）- 用于点赞、收藏激活状态
- **背景色**：纯黑 `#000000` - 沉浸式体验
- **文字色**：白色系 `#FFFFFF` / `rgba(255,255,255,0.7)`

#### 2.2 字体系统
- 主字体：PingFang SC, Helvetica Neue, Arial, sans-serif
- 标题：18px, 粗体 (w700)
- 简介：14px, 常规 (w400)
- 标签：12px, 中等 (w500)

#### 2.3 动画效果
- 视频切换：垂直滑动过渡
- 点赞动画：心形放大 + 飞行效果
- 进度条：渐变显示/隐藏
- 按钮反馈：缩放 + 颜色变化

### 3. 手势交互设计

| 手势 | 功能 | 视觉反馈 |
|------|------|----------|
| 上/下滑动 | 切换视频 | 页面跟随手指移动 |

| 双指缩放 | 视频缩放 | 画面实时响应 |

### 5. 响应式设计

#### 5.1 手机竖屏
- 全屏视频播放
- 右侧操作按钮半透明叠加
- 底部信息区渐变叠加
- 底部导航栏固定

#### 5.2 平板/横屏
- 视频居中显示
- 操作按钮移至右侧边缘
- 信息区宽度限制
- 导航栏自动隐藏

#### 5.3 TV 模式
- 大字体、大按钮
- D-pad 导航支持
- 自动隐藏控制栏
- 长时间无操作暂停

### 6. PWA 特性

- 可安装到主屏幕
- 离线缓存支持
- 推送通知
- 后台播放
- 画中画模式

## 可借鉴的设计模式

### 1. 键盘快捷键系统

**EmbyX 实现**：
- 使用全局键盘监听
- 支持多种按键组合
- 提供快捷键帮助面板

**建议实现**：
```dart
// 在 FeedView 中添加键盘监听
void _handleKeyEvent(KeyEvent event) {
  if (event is KeyDownEvent) {
    switch (event.logicalKey) {
      case LogicalKeyboardKey.keyW:
      case LogicalKeyboardKey.arrowUp:
        _goToPreviousVideo();
        break;
      case LogicalKeyboardKey.keyS:
      case LogicalKeyboardKey.arrowDown:
        _goToNextVideo();
        break;
      case LogicalKeyboardKey.space:
        _togglePlayPause();
        break;
      // ... 更多快捷键
    }
  }
}
```

### 2. 视图切换机制

**EmbyX 实现**：
- 快捷键 E 切换视图
- 海报墙视图：网格布局、分页、随机
- 视频流视图：TikTok 风格垂直滑动

**建议实现**：
- 添加 `ViewMode` 枚举：`feed` / `poster`
- 使用 `PageView` 实现视图切换动画
- 海报墙使用 `GridView.builder`

### 3. 媒体库快速选择

**EmbyX 实现**：
- 快捷键 G 打开媒体库选择器
- 底部弹出面板
- 支持搜索和过滤

**建议实现**：
- 添加 `LibrarySelector` 组件
- 使用 `ModalBottomSheet` 展示
- 支持键盘导航

### 4. 视频信息面板

**EmbyX 实现**：
- 快捷键 V 显示媒体信息
- 包含：编码、分辨率、码率、时长

**建议实现**：
- 添加 `VideoInfoPanel` 组件
- 从 `VideoPlayerController` 获取元数据
- 半透明叠加显示

### 5. 主题颜色优化

**EmbyX 颜色方案**：
```css
primary: '#52B54B'    /* Emby 绿 - 关注、进度 */
secondary: '#EE3152'  /* 抖音红 - 点赞、收藏 */
```

**EmbyTok 当前颜色**：
```dart
primaryPink: Color(0xFFE91E63)  /* 粉红 - 点赞、收藏 */
```

**建议**：保持当前粉色主题，但可以添加 Emby 绿作为辅助色用于关注按钮。

## 实施优先级

### 高优先级
1. **键盘快捷键系统** - 提升桌面端体验
2. **视图切换** - 海报墙与视频流切换

### 中优先级
3. **媒体库快速选择** - 提升导航效率
4. **顺序/随机切换** - 增加播放模式
5. **PWA 优化** - 安装体验和离线支持

### 低优先级
6. **视频宽高比切换** - 特定场景需求
7. **媒体信息面板** - 调试和高级用户
8. **画中画模式** - 多任务场景
