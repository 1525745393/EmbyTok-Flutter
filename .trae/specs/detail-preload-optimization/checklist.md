# 视频详情页优化 & 预加载优化 - 验证清单

## 详情页 - 相关推荐

- [ ] 详情页底部显示「相关推荐」标题
- [ ] 相关推荐为横向滚动卡片列表
- [ ] 推荐卡片显示封面图 + 标题
- [ ] 点击推荐卡片跳转到对应详情页
- [ ] 加载中显示骨架屏/加载状态
- [ ] 加载失败不阻塞页面其他内容
- [ ] 最多显示 12 条推荐

## 详情页 - 元信息增强

- [ ] 显示导演信息（如有）
- [ ] 显示视频时长（格式化显示）
- [ ] 信息布局美观，不拥挤
- [ ] 没有信息时不显示对应标签

## 预加载 - 封面图

- [ ] 切换到当前视频时，预加载下一个视频的封面图
- [ ] 滑动到下一个视频时，封面图无明显闪烁
- [ ] 预加载不影响当前视频播放

## 预加载 - 策略优化

- [ ] 预加载缓存有数量上限
- [ ] 远离当前位置的预加载被及时清理
- [ ] 快速滑动后内存不无限增长
- [ ] 切换 feedType 时清理旧的预加载缓存
